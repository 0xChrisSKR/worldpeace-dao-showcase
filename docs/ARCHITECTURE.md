# Architecture

WorldPeace DAO is structured as a public Web3 DApp frontend showcase.

## Diagram

- `assets/diagrams/architecture.svg`
- `assets/diagrams/architecture.mmd`

Core areas:

- Wallet connection
- BSC / BNB Smart Chain interaction
- Governance interface
- Treasury interface
- Charity DEX concept
- Guild / token-gated community logic
- Chat entry
- Whitepaper route
- Unity WebGL game route
- Public demo website

The architecture focuses on product flow rather than custody. The project does not custody private keys and does not claim production donation volume or TVL.

This repository is intentionally a public showcase. It does not include source folders, private configuration, deployment secrets, environment files, or git history from the implementation repository.
