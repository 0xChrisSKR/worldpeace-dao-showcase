# 104 Project Summary

WorldPeace DAO 是一個 on-chain charity DApp 與治理平台原型，結合 WorldPeace Wallet、DAO Governance、Treasury、Charity DEX、Guild、Chat、Whitepaper 與 Unity WebGL / Start Game 入口。這個專案的目標，是探索如何用 Web3 產品設計支援小額捐款概念、社群提案、鏈上治理、財務透明與 token-gated community。

我負責產品規劃、DApp 介面設計、錢包連線流程、治理與 treasury 產品流程、Charity DEX 概念、白皮書內容整理，以及 Unity WebGL 遊戲入口整合。技術上使用 Next.js、React、TypeScript、Tailwind CSS、Wagmi、Viem、WalletConnect / Reown、BSC / BNB Smart Chain、Unity WebGL 與 Vercel。

此專案不宣稱實際捐款金額、TVL、活躍 DAO 用戶數、投資報酬或已完成真實慈善執行；它是公開展示用的 Web3 DApp 產品原型，也成為後續 C-Wallet、C-Chain、TRACE ProofFeed 與 TRACE AI Platform 的早期產品脈絡。
